📥 Import de données

1. Exportez votre fichier Excel en CSV (séparateur ;)

2. Allez dans Import / Export

3. Sélectionnez la base cible (Bureau, Portables…)

4. Glissez ou sélectionnez le fichier CSV

5. Vérifiez l'aperçu et confirmez l'import

⚠️ Les doublons sont détectés automatiquement par N° de série.

🖨️ Impression & Export

Export CSV : compatible Excel, encodage UTF-8

Export Word (.doc) : tableau prêt à l'impression

Impression listes : filtrer par :

  • Type d'équipement
  • Genre (♀ Féminin / ♂ Masculin)
  • Sous-direction / Service
  • Date d'obtention
⚠️ Important : Les données sont sauvegardées localement dans votre navigateur (localStorage). Pour sauvegarder de manière permanente, utilisez l'export CSV régulièrement.
